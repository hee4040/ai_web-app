# Phase 1 / Phase 2 분리 PRD

---

## 1. 서비스 비전

> **“파편화된 개발 환경 설정 경험을 넘어,  
> 사례와 데이터로 최적의 설정 방법을 연결하는 개발자 지식 허브”**

본 서비스는 개발자 · 대학생 · 취준생이  
자신의 **OS · 도구 · 버전 환경**에 맞는  
**실제 환경 설정 성공 사례**를 빠르게 찾고 비교할 수 있도록 돕는다.

단일한 정답을 제시하는 것이 아니라,  
여러 사용자의 시행착오와 트러블슈팅 경험을 구조화하여  
개발 환경 설정 과정에서 발생하는 **반복적인 실패를 줄이는 것**이 목표다.

---

## 2. Phase 1 — 수업 내 구현 목표 (MVP)

### 목표

> **“로그인 → 레시피 작성 →  
> 카테고리 탐색 → 정렬된 리스트 확인 →  
> 상세 확인 → AI 보조 결과 확인”**

이 **완결된 정보 탐색 흐름**을  
**3일 안에 구현**하는 것이 Phase 1의 목표다.

---

## 2.1 인증 및 유저 관리 (Phase 1)

### 포함 기능

- Google Social Login
- Supabase Auth 기반 인증
- 로그인 사용자 식별 (`user_id`)

### 의도

- 인증은 서비스 핵심 기능이 아닌 **전제 조건**
- 가장 표준적이고 구현이 빠른 방식만 사용

---

## 2.2 환경 설정 레시피 관리 (Recipe CRUD – Phase 1)

### 포함 기능

- **레시피 작성**
  - 제목
  - 간단한 설명
  - 카테고리 (단일 선택)
  - 환경 태그 (텍스트 배열)
  - Step-by-Step 설정 과정 (텍스트 기반 단계 목록)
  - Step당 이미지 0~1장 업로드 (Supabase Storage)
- 레시피 리스트 조회
- 레시피 상세 조회
- 레시피 삭제 (작성자 본인만 가능)

### 의도

- “구조가 보이는 레시피”까지만 구현
- 이미지 다중 업로드, Drag & Drop, 고급 에디터는 Phase 2로 이관

---

## 2.3 탐색 기능 (Phase 1)

### 포함 기능

- **카테고리별 리스트 보기**
  - 예: 전체 / Git / Docker / ROS
- DB 쿼리 기반 필터링

---

## 2.4 트러블슈팅 입력 + AI 보조 (Phase 1)

### 포함 기능

- 트러블슈팅 텍스트 입력
- AI 호출 (작성 시 1회)
  - 에러 요약
  - 핵심 키워드 추출
- AI 결과를 DB에 저장 후 화면에 표시

### 중요 원칙

- 리스트/상세 진입 시 AI 재호출 ❌
- 목표는 **AI 연동 경험 + 결과를 데이터로 저장**하는 것

---

## 2.5 UI / UX (Phase 1)

### 페이지 구성

- **메인 페이지**
  - 카테고리 탭
  - 정렬 옵션
  - 레시피 리스트
- **상세 페이지**
  - 레시피 내용
  - 트러블슈팅 요약 결과
- **작성 페이지**
  - 단일 폼 구조

---

## 2.6 Phase 1 성공 기준 (Definition of Done)

- Google 로그인 가능
- 레시피 작성 → 카테고리별 리스트 노출 가능
- 최신순 정렬 정상 동작
- 레시피 상세 페이지 진입 가능
- AI 요약 결과가 DB에 저장되고 화면에 표시됨
- **전체 사용자 흐름이 끊기지 않음**

---

## 3. Phase 2 — 수업 이후 고도화 단계

### 3.1 레시피 구조 고도화

- 단계별 테이블 분리
- 이미지 여러 장 업로드
- Drag & Drop 정렬

---

### 3.2 트러블슈팅 고도화

- 여러 사용자 기여
- 키워드 자동 매칭
- 유사 에러 묶음 보기

---

### 3.3 검색 / 필터링 고도화

- 다중 태그 필터
- 정렬 알고리즘 고도화
- **정렬 기능**
  - 최신순 (`created_at DESC`)
  - 공감순
- **단순 검색**
  - 제목 기준 키워드 검색
  - `ILIKE` 기반 단일 검색
- 환경 유사도 기반 추천

---

### 3.4 인터랙션 및 개인화

- 북마크
- 공감(좋아요)
- 마이페이지
- 공개 / 비공개 관리

---
